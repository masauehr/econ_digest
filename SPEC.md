### 特に注目しているトピック（優先収集・必ず記事に含める）

#### 1. 日本の株式市場・マーケット動向
- **対象**: 日経平均・TOPIX・個別株・業種別指数
- **注目点**: 週次騰落率・出来高・外国人投資家動向・機関投資家動向
- **ツール**: Bloomberg Terminal相当のWebデータ
- **キーワード**: `日経平均 今週`, `TOPIX 週次`, `東証 相場観`, `外国人買い越し売り越し`

#### 2. 為替・金利市場
- **対象**: ドル円・ユーロ円・金利（長短国債利回り）・日米金利差
- **注目点**: 日銀の為替介入リスク・FRBの利下げ見通し・金利スプレッド
- **キーワード**: `ドル円 推移 今週`, `日米金利差 今週`, `為替介入 リスク`

#### 3. 日本銀行・金融政策
- **対象**: 政策金利・国債買い入れ・YCC（長短金利操作）・展望レポート
- **注目点**: 次回政策決定会合の見通し・市場の利上げ織り込み度
- **情報源（直接確認）**:
  - 日本銀行: https://www.boj.or.jp/
  - 日銀総裁会見要旨
- **キーワード**: `日銀 政策金利 最新`, `日銀 金融政策決定会合 今週`, `植田総裁 発言`

#### 4. 国内経済指標・統計
- **対象**: GDP速報・消費者物価（CPI）・鉱工業生産・機械受注・貿易統計・雇用統計
- **注目点**: 前期比・前年比・市場予想との乖離
- **情報源（直接確認）**:
  - 内閣府: https://www.esri.cao.go.jp/
  - 総務省統計局: https://www.stat.go.jp/
  - 財務省: https://www.mof.go.jp/
- **キーワード**: `GDP 速報 今期`, `CPI 消費者物価 今月`, `貿易収支 今月`, `完全失業率 最新`

#### 5. 米国・海外経済
- **対象**: 米国GDP・雇用統計（NFP）・CPI・FRB利下げ見通し・中国経済指標
- **注目点**: FRBの金融政策変更が円相場・日本株に与える影響
- **キーワード**: `米国 雇用統計 今月`, `FRB 利下げ 見通し 最新`, `中国 PMI 景況感`

#### 6. 企業業績・M&A・産業動向
- **対象**: 主要企業の決算発表・業績修正・M&A・設備投資
- **注目点**: 業績修正の上下振れ・セクター別トレンド
- **注意**: 特定銘柄への投資推奨と受け取られないよう、あくまでニュースの概観にとどめる
- **キーワード**: `決算 業績修正 今週 日本`, `M&A 企業買収 今週`, `設備投資 日本 最新`

## 自動実行フロー

```
launchd（毎週金曜 09:00）
  └─ run_econ_ollama.sh
       ├─ 日付判定（第1金曜か否か）
       ├─ claude CLI 起動 / Ollama 起動
       │    └─ CLAUDE.md の指示に従い記事生成
       │         ├─ WebSearch で最新情報を収集
       │         ├─ 記事ファイルを生成（Write）
       │         └─ README.md のリンク一覧を更新（Edit）
       └─ git add / commit / push
```

## 品質基準

- 情報ソースのURLを必ず記載する
- 株価・為替は「〜前後」「〜水準」など幅を持たせた表現を用いる（投資助言に見えないようにする）
- 推測・憶測は「〜とみられる」「〜の可能性がある」と明示する
- 週次まとめ: 最低5トピック以上（国内経済動向2件以上必須）
- 月次まとめ: 最低3つの主要トピックを深掘り＋月間パフォーマンス表必須
- 日本語で記述（経済指標名・指数名は原語も記載）
